Basic

WPUI adalah kumpulan UI component untuk WordPress theme. Copy-paste, tanpa framework, tanpa jQuery. Developer cukup ambil kode yang mereka butuhkan.

Ringan, accessible, dan konsisten. Setiap component menggunakan pola HTML dan JS yang sama sehingga mudah dipelajari dan diprediksi.

Gratis selamanya. Open source. Tidak ada lisensi, tidak ada subscription, tidak ada catch.
